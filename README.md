# Eterna Website

**Eterna Website** est un site web moderne, responsive et complet conçu pour présenter un large éventail de contenus professionnels et créatifs. Développé avec **HTML, CSS et Bootstrap 5**, il offre une expérience utilisateur fluide et attractive, adaptée à tous les types d’appareils.

### Structure et Sections Principales

Le site comprend plusieurs sections importantes, accessibles via une navigation claire et intuitive :

* **Home** : page d’accueil avec présentation visuelle forte et messages clés.
* **About** : description détaillée de l’entreprise ou du profil.
* **Services** : présentation des services offerts, accompagnée d’icônes et d’explications.
* **Portfolio** : galerie de 12 projets répartis en catégories (App, Product, Branding, Books) avec filtrage fluide basé sur CSS uniquement, animations et effets hover sur les images.
* **Team** : présentation des membres de l’équipe avec leurs rôles et photos.
* **Pricing** : tableaux ou sections tarifaires clairs et attractifs.
* **Blog** : espace dédié aux articles, actualités ou contenus éditoriaux.
* **Dropdown** : menu déroulant pour des options supplémentaires.
* **Contact** : formulaire de contact ou informations pour joindre facilement l’équipe.

### Fonctionnalités Clés

* **Filtrage du portfolio CSS-only** : sélection fluide des catégories avec transition animée des projets affichés.
* **Effets hover dynamiques** sur les images du portfolio, révélant des icônes d’action pour visualiser ou accéder aux détails.
* **Design responsive** garanti par Bootstrap 5, adapté à toutes les tailles d’écran.
* **Navigation claire et ergonomique** avec menu complet pour faciliter l’accès à toutes les sections.
* **Animations et transitions CSS** soignées pour améliorer l’expérience utilisateur sans alourdir le site.

### Technologies Utilisées

* HTML5 pour une structure sémantique claire.
* CSS3 pour le style, les animations et les transitions.
* Bootstrap 5 pour la mise en page responsive et les composants UI.
* Bootstrap Icons pour les icônes interactives.
